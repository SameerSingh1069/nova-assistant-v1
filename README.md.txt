# NOVA Assistant (V1 - Building Block)

Desktop AI Assistant built using:

- Python
- Tkinter
- Ollama
- Global Hotkey

## Features
- Ctrl + Alt + A toggle
- Open apps and websites
- Local AI via Ollama

## Run Instructions

1. Install Python
2. Install Ollama
3. Run:
   ollama serve
4. Install dependencies:
   pip install -r requirements.txt
5. Run:
   python assistant.py
